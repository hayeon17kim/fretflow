# FretFlow — 시니어 BM/PM 종합 분석 리포트

**분석일:** 2026-02-16
**분석자:** Senior BM / Senior PM 관점
**대상:** FretFlow (기타 프렛보드 학습 앱)

---

## 1. Executive Summary

FretFlow는 **SM-2 스페이스드 리피티션 알고리즘**을 핵심으로 한 기타 프렛보드 학습 앱이다. "Anki for Guitar"라는 명확한 포지셔닝을 가지고 있으며, Expo/React Native 기반 크로스플랫폼 모바일 앱으로 구현되어 있다.

**현재 상태를 한 문장으로:** 핵심 학습 엔진(SM-2)과 Lv.1/Lv.4 퀴즈 파이프라인은 완성되어 있으나, Lv.2/Lv.3이 여전히 목업 상태이고, 리텐션을 만드는 핵심 루프(세션 완료 → 성취감 → 복귀)가 빠져 있어 **MVP 직전 단계**에 머물러 있다.

### 2.2 4-Level 커리큘럼 완성도

```
Lv.1 음이름 맞추기  ████████████ 100% — SM-2 연동, 동적 카드 생성, 완전 동작

Lv.3 스케일 패턴    ███░░░░░░░░░  25% — UI 완성, MOCK 1문제, SM-2 미연동
Lv.4 이어트레이닝   ████████████ 100% — SM-2 연동, 동적 카드 생성 (오디오 파일 확인 필요)
```

### 2.3 핵심 문제점 (Critical Bugs & Gaps)

**🔴 P0 — 제품이 동작하지 않는 수준의 문제**

1. **Lv.2/Lv.3 학습 데이터가 사라진다**
   - `interval.tsx`와 `scale.tsx`에 `useSpacedRepetition`이 import되지 않음
   - 사용자가 풀어도 MMKV에 저장되지 않아 다음 세션에 반영 안 됨
   - 홈 화면의 "오늘의 복습" 카운트에도 잡히지 않음

2. **todayStats가 앱 재시작 시 초기화**
   - Zustand store가 persist 미적용 → 앱 껐다 키면 streak=0, cardsReviewed=0
   - 사용자의 "연속 학습 기록"이 무의미해짐

3. **incrementReview()가 어디서도 호출되지 않음**
   - 퀴즈 완료 후 todayStats.cardsReviewed가 증가하지 않음
   - 홈 화면의 모든 통계가 항상 0

**🟡 P1 — 사용자 경험을 심각하게 저해하는 문제**

4. **세션 완료 화면 없음** — 10문제 풀고 `router.back()`으로 그냥 돌아감. 성취감 0.
5. **"복습 시작" CTA가 항상 note 퀴즈로만 이동** — due 카드가 어떤 레벨에 있든 무시
6. **dailyGoal 설정은 있지만 아무 곳에서도 사용되지 않음**
7. **vibrationEnabled 설정은 있지만 실제 진동이 없음**
8. **카드 ID에 Date.now() 포함** — 같은 음/같은 문제가 매번 새 카드로 생성되어 SM-2 학습 곡선이 제대로 누적되지 않음

**🟢 P2 — 개선하면 좋은 부분**

9. **진도율 계산 불일치** — 홈 화면은 `전체 카드 수/60`, `getLevelProgress()`는 `마스터 카드 수/60`
10. **Fretboard 온보딩 텍스트가 하드코딩 한국어** — i18n 미적용

## 3. 시장 환경 분석

### 3.1 시장 규모

- **기타 학습 앱 시장:** 2023년 $3.25억, CAGR 3% 성장 (→ 2033년 예상)
- **온라인 음악 교육:** 2025년 $33.2억, CAGR 14.8% (→ 2030년 $66.3억)
- **아시아태평양 성장률:** CAGR 17.1%로 가장 빠르게 성장
- **앱 기반 인터페이스가 전체 매출의 51.3%** — 모바일 퍼스트 전략 적합

### 3.2 경쟁 환경 매핑

| 경쟁자 | 포지셔닝 | 강점 | 약점 | 가격 |
|--------|----------|------|------|------|
| **Yousician** | 게이미피케이션 + 마이크 감지 | 멀티 악기, 높은 완성도 | 비쌈, 프렛보드 이론 약함 | $13.99/월 |

### 3.3 FretFlow의 시장 포지션

**FretFlow가 위치해야 할 블루오션:**

기존 앱들은 두 가지 축으로 나뉜다:
- **"Play Along" 앱** (Yousician, Simply Guitar): 곡 따라치기 → 실용적이지만 이론/프렛보드 이해 부족
- **프렛보드 트레이너** (Fret Pro, FretMemo): 음 암기 → 이론적이지만 재미없고 단순 반복

FretFlow는 **"체계적 프렛보드 마스터리"** 라는 독보적 포지션을 가짐:
- 음이름 → 인터벌 → 스케일 → 이어트레이닝의 4단계 커리큘럼
- SM-2로 과학적 복습 주기
- 프렛보드 인터랙션 기반 (단순 객관식 아님)

**이 포지션의 문제:** Fret Pro가 이미 스페이스드 리피티션 + 프렛보드를 하고 있음. 차별화를 위해서는 **커리큘럼 깊이와 UX 품질**로 승부해야 한다.

## 4. PMF (Product-Market Fit) 진단

### 4.1 타겟 사용자

**Primary:** 기타 초중급자 (6개월~2년차) 중 "프렛보드를 외우고 싶은" 사람
- 코드만 치다가 솔로/즉흥연주로 넘어가고 싶은 단계
- "프렛보드가 안 외워져서" 정체된 기타리스트

**Secondary:** 음악 이론을 체계적으로 공부하고 싶은 기타리스트

### 4.2 핵심 가치 제안

> "매일 10분, 과학적으로 프렛보드를 마스터하세요"

이 가치 제안이 성립하려면:
- ✅ "매일 10분" → 세션 크기 10문제 × 4~6초/문제 ≈ 적절
- ❌ "과학적으로" → SM-2가 2/4 레벨에서 미작동
- ❌ "마스터" → 레벨 잠금 해제 로직은 있으나, 실제로 Lv.2/3에서 학습이 누적되지 않음

### 4.3 PMF 달성을 위한 최소 조건

1. **4개 레벨 모두 SM-2 파이프라인 완성** (현재 50% 완성)
2. **세션 완료 → 성취감 루프** (현재 0% 완성)
3. **일일 목표 + 스트릭 시스템 실제 동작** (현재 설정만 있고 미동작)

## 6. 수익 모델 제안

### Freemium + Subscription

```
무료 티어:
├── Lv.1 음이름 퀴즈 (무제한)
├── Lv.2~4 미리보기 (하루 3문제)
├── 기본 통계
└── 광고 (배너, 세션 간 전면)

프리미엄 ($4.99/월, $39.99/년):
├── 전 레벨 무제한
├── SM-2 기반 맞춤 복습
├── 상세 학습 분석
├── 클라우드 동기화
├── 광고 제거
└── 마이크 연동 (Phase 3)
```

## 9. 최종 권고

### 지금 당장 해야 할 3가지

1. **Phase 1을 이번 주 안에 끝내라.** Lv.2/3 SM-2 연동 + 카드 ID 수정 + todayStats 영속화. 이것 없이는 앱이 "동작하지 않는" 상태다.

2. **세션 완료 화면을 만들어라.** 사용자가 10문제를 풀고 `router.back()`으로 돌아가는 것은 "성취감 제로"다. 이것이 리텐션의 핵심이다. 정답률, 스트릭 카운트, "내일 N장 복습 예정" 메시지만으로도 충분하다.

3. **한국 시장부터 작게 검증하라.** 이미 한국어가 완벽히 지원되고, 한국 기타 커뮤니티(네이버 카페, 에펨코리아, 기타 관련 유튜브)에서 소규모 테스트를 진행하라. 글로벌보다 한국에서의 초기 PMF 검증이 더 빠르고 효율적이다.
