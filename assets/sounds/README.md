# 귀 훈련 오디오 파일

이 디렉토리에는 귀 훈련 퀴즈에서 사용할 기타 개방현 소리 파일이 필요합니다.

## 필요한 파일

다음 5개 파일을 이 디렉토리에 추가해주세요:

- `E.mp3` - 1번 줄 개방현 (높은 E, 330Hz)
- `A.mp3` - 5번 줄 개방현 (A, 110Hz)
- `D.mp3` - 4번 줄 개방현 (D, 147Hz)
- `G.mp3` - 3번 줄 개방현 (G, 196Hz)
- `B.mp3` - 2번 줄 개방현 (B, 247Hz)

## 오디오 파일 준비 방법

### 옵션 1: 직접 녹음
1. 실제 기타로 각 개방현을 녹음
2. mp3 형식으로 변환
3. 파일명을 위와 같이 변경

### 옵션 2: 무료 샘플 다운로드
다음 사이트에서 무료 기타 샘플을 다운로드할 수 있습니다:

- **Freesound** (https://freesound.org/)
  - 검색어: "guitar open string E", "guitar open A" 등

- **Philharmonia Orchestra** (https://philharmonia.co.uk/resources/sound-samples/)
  - 악기 샘플 무료 제공

- **Pianobook** (https://www.pianobook.co.uk/)
  - 다양한 악기 샘플

### 옵션 3: 온라인 톤 생성기 사용
1. **Online Tone Generator** (https://onlinetonegenerator.com/)에서 각 주파수 생성:
   - E4: 330Hz
   - B3: 247Hz
   - G3: 196Hz
   - D3: 147Hz
   - A2: 110Hz
2. WAV로 다운로드 후 mp3로 변환
3. 파일명 변경

## 파일 규격 권장사항

- 형식: MP3
- 길이: 2-3초
- 비트레이트: 128kbps 이상
- 샘플레이트: 44.1kHz

## 참고사항

오디오 파일이 없으면 귀 훈련 퀴즈가 정상적으로 작동하지 않습니다.
재생 버튼을 눌렀을 때 콘솔에 경고 메시지가 표시됩니다.
